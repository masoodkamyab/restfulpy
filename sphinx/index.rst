########
restfulpy
########

A tool-chain for creating restful web applications.

********
Features
********

- Create automatically metadata for API 
- Migration using alembic
- Advanced wrapper for logging
- Validations base on models
- Wrapper for SQLAlchemy ORM
- Powerfull CLI tools
- Wrapper for mokup server
- Wrapper for testing
- Encapsulating all stuff about authentication
